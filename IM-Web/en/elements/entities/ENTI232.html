Entity

Domain

Description of the value range of one (or more) attribute(s). The types numerical, text, time or enumeration are to be understood conceptually. The real data types and, if applicable, formats (for numbers and times) for storage are defined in the implementation. Value groups are attribute groups, i.e. several attributes that occur more than once together are combined under a new name for easier handling. Example telephone number consisting of country code and telephone number

Entity

Key element

Group of identifying relationships and/or attributes of an entity.
Each combination of values in a key is unique to the entity.
I.e. it identifies exactly 1 entry in the entity.
The sequence number is an aid.
Strictly speaking, only the entity + the set of all key elements is unique for a key.

Group Attribute

There are attributes that consist of several components, but can be regarded as one attribute for an entity.
The components of this attribute group always occur together.
The fact that an attribute contains partial values in which one is interested separately basically violates the 1st normal form. However, if the elements of an attribute group are attributes themselves and the group can be understood as a placeholder for several attributes, this violation can be resolved.

Examples: Hotel room no. consisting of floor and room no.
(ATTENTION: not 1st digit = floor, but 1 field floor and 1 field room no.) Flight identification consisting of airline short and flight number (LX 247)

Synonym Gruppenattribut
Description Es gibt Attribute, die aus mehreren Bestandteilen bestehen, aber für eine Entität wie ein Attribut angesehen werden (können). Die Bestandteile dieser Attributgruppe treten immer gemeinsam auf. Dass ein Attribut Teilwerte enthält, für die man sich separat interessiert, verletzt im Grundsatz die 1. Normalform. Wenn aber die Elemente einer Attributgruppe selber Attribute sind und die Gruppe als Platzhalter von mehreren Attributen verstanden werden kann, ist diese Verletzung auflösbar.
